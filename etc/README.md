## ゲーム概要
制限時間内でブロックを横に3つ並べて消してスコアを稼ぐパズルゲームです。

## 実行方法
1. ソースをコンパイルする  
   `javac -encoding UTF-8 Bustra.java`

2. 実行する  
   `java Bustra`
